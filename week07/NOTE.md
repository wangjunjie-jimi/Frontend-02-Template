## CSS

### 盒模型

- css选择器选中的是元素
- css选择器选中的元素，在排版时可能产生多个盒
- 排版和渲染的基本单位是盒
- box-sizing
  - content-box
  - border-box 

### 正常流
- 收集盒进行
- 计算盒在行中的排布
- 计算行的排布
- 行级排布
  - line-top text-top base-line text-bottom line-bottom
- 块级排布
- margin折叠
  - 只发生在BFC中
- BFC合并
  - Block Container
    - block
    - inline-block
    - table-cell
    - flex item
    - grid cell
    - table-caption
  - Block-level Box
  - 影响
    - float 
    - 边距折叠
